## Core Features

### 1. Daily Nutrition
- [ ] Calorie + macro tracker (protein, carbs, fat)
- [ ] Meal logging (4 meals default structure)
- [ ] **Indian food database** — roti, dal, rice, chicken, paneer, curd, biryani portions
- [ ] Barcode scan for packaged foods / supplements
- [ ] Quick-add: whey scoop, eggs (count), chicken (grams)

### 2. Micronutrients
- [ ] Vitamin D, B12, magnesium, calcium, iron, zinc
- [ ] Supplement log with reminders (D3 weekly alert)

### 3. Body Progress
- [ ] Daily weight (morning, fasted)
- [ ] Weekly progress photos (front/side/back)
- [ ] Optional: measurements (chest, arms, waist, thighs)

### 4. Training Integration
- [ ] Tag days: boxing / upper / lower / rest
- [ ] Auto-adjust calorie target by day type

### 5. Supplements Module
- [ ] Daily checklist: creatine, B12, fish oil, magnesium, K2
- [ ] Weekly: D3 60,000 IU reminder
- [ ] Stock / refill alerts

### 6. Dashboard
- [ ] Today's macros vs target (progress bars)
- [ ] Protein streak counter
- [ ] 3-month goal: 72 → 77 kg tracker
